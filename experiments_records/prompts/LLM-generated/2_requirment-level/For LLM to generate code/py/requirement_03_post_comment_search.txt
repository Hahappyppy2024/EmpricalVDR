## Functional Requirements (function-only)

### E) Posts (any logged-in user within course context)

* FR-P1 create_post()

  * HTML: GET /courses/:course_id/posts/new, POST /courses/:course_id/posts
  * API:  POST /api/courses/:course_id/posts {title, body}
* FR-P2 list_posts()

  * HTML: GET /courses/:course_id/posts
  * API:  GET /api/courses/:course_id/posts
* FR-P3 get_post()

  * HTML: GET /courses/:course_id/posts/:post_id
  * API:  GET /api/courses/:course_id/posts/:post_id
* FR-P4 update_post()

  * HTML: GET /courses/:course_id/posts/:post_id/edit, POST /courses/:course_id/posts/:post_id
  * API:  PUT /api/courses/:course_id/posts/:post_id {title, body}
* FR-P5 delete_post()

  * HTML: POST /courses/:course_id/posts/:post_id/delete
  * API:  DELETE /api/courses/:course_id/posts/:post_id

### F) Comments (any logged-in user within course context)

* FR-CM1 create_comment()

  * HTML: POST /courses/:course_id/posts/:post_id/comments
  * API:  POST /api/courses/:course_id/posts/:post_id/comments {body}
* FR-CM2 list_comments()

  * HTML: GET /courses/:course_id/posts/:post_id/comments (optional but preferred)
  * API:  GET /api/courses/:course_id/posts/:post_id/comments
* FR-CM3 update_comment()

  * HTML: POST /courses/:course_id/posts/:post_id/comments/:comment_id
  * API:  PUT /api/courses/:course_id/posts/:post_id/comments/:comment_id {body}
* FR-CM4 delete_comment()

  * HTML: POST /courses/:course_id/posts/:post_id/comments/:comment_id/delete
  * API:  DELETE /api/courses/:course_id/posts/:post_id/comments/:comment_id

### G) Search (any logged-in user within course context)

* FR-S1 search_posts()

  * HTML: GET /courses/:course_id/search?keyword=...
  * API:  GET /api/courses/:course_id/search/posts?keyword=...
* FR-S2 search_comments()

  * HTML: GET /courses/:course_id/search/comments?keyword=...
  * API:  GET /api/courses/:course_id/search/comments?keyword=...
