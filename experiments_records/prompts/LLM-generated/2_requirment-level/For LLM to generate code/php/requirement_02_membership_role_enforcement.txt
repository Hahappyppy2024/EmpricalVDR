## Functional Requirements (function-only)

### D) Membership (Membership v0)

* FR-M1 add_member()

  * HTML: GET /courses/:course_id/members/new, POST /courses/:course_id/members
  * API:  POST /api/courses/:course_id/members {user_id, role_in_course}
* FR-M2 list_members()

  * HTML: GET /courses/:course_id/members
  * API:  GET /api/courses/:course_id/members
* FR-M3 update_member_role()

  * HTML: POST /courses/:course_id/members/:membership_id
  * API:  PUT /api/courses/:course_id/members/:membership_id {role_in_course}
* FR-M4 remove_member()

  * HTML: POST /courses/:course_id/members/:membership_id/delete
  * API:  DELETE /api/courses/:course_id/members/:membership_id
* FR-M5 my_memberships()

  * HTML: GET /memberships
  * API:  GET /api/memberships
